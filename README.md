# DragonSkins Mod

A Fabric mod for Minecraft that adds custom skins, capes, and nametag rendering.

## Features

- Custom skin loading from DragonClient API
- Custom cape support
- Enhanced nametag rendering with badges
- Multi-version support (1.16.2 - 1.21.4)

## Building

```bash
./gradlew build
```

## Installation

1. Install Fabric Loader
2. Install Fabric API
3. Drop the mod JAR into your mods folder
